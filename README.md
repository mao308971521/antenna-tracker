# 天线行业情报追踪系统

实时追踪天线行业市场动态、企业信息、价格走势、标准更新。

## 功能模块

- 📊 **市场概览**：全球及中国市场规模、增长率、预测数据
- 📈 **细分市场**：5G天线、汽车天线、船舶天线等细分领域
- 📰 **行业动态**：新闻、报告、集采信息
- 🏭 **重点企业**：国内外主要厂商信息
- 💰 **原材料价格**：铜、铝、PCB等价格走势
- 📋 **行业标准**：国标、行标、国际标准更新

## 技术栈

- Next.js 14
- React 18
- TypeScript
- 纯静态导出（无需服务器）

## 本地开发

```bash
npm install
npm run dev
```

## 更新数据

编辑 `data/` 目录下的 JSON 文件，然后重新构建：

```bash
npm run build
```

## 部署

本项目已配置为静态导出，可直接部署到：
- Vercel
- Netlify
- GitHub Pages
- 任意静态托管服务

## 数据来源

- 中商产业研究院
- 产业调研网
- 腾讯网、搜狐等行业媒体
- 3GPP等国际标准组织
